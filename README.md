# Blog-SpringBoot
# blog

**简介：** 基于Spring Boot开发的小型个人博客+后台管理

**技术组合：**

* 后端：Spring Boot + Mybatis + thymeleaf模板
* 数据库：MySQL5.7+Redis
* 前端UI：Semantic UI框架

**工具与环境：**

* IDEA
* Maven3
* JDK8
